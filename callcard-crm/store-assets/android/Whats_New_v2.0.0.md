# What's New — v2.0.0 (Play Store)

Paste this into Play Console → Production → New release → Release notes.

Cross-device sync, rebuilt.

• Customer data syncs through iCloud (iOS) or your Google account (Android) — edit anywhere, see it everywhere within seconds
• Offline edits are preserved when another device syncs first
• Deleted customers stay deleted everywhere (no more ghosts)
• Per-record sync — no all-or-nothing overwrites
• Square login auto-refreshes so you stay connected longer
• Faster customer search, more reliable calendar events, lots of polish
